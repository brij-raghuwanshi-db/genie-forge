# Development Documentation

This folder contains internal development and testing documentation for Genie-Forge.

| File | Description |
|------|-------------|
| `TEST_GAP_ANALYSIS.md` | Comprehensive analysis of missing test cases and edge cases |

The user-facing documentation is in the parent `docs/` folder.
